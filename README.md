# Hallucination Detection in Large Language Models

## 📖 Overview
This project focuses on detecting hallucinations in Large Language Models (LLMs) using multiple techniques such as uncertainty estimation, self-consistency, retrieval-based verification, Natural Language Inference (NLI), and semantic similarity.

## 🚀 Features
- Uncertainty-based hallucination detection
- Self-consistency checking
- Retrieval-Augmented Verification (RAG)
- NLI-based contradiction detection
- Semantic similarity analysis
- Benchmark evaluation on public datasets
